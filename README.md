## 🛠️ Tech

This is a **static single-page landing site** built with:
- HTML5 + [Tailwind CSS](https://tailwindcss.com/) (via CDN)
- [Material Symbols](https://fonts.google.com/icons) icons
- [Inter](https://fonts.google.com/specimen/Inter) font
- Zero build step — just open `index.html`
